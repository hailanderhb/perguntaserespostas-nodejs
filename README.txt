Readme:

Backend: 

Uma aplicação em nodejs + express no backend para controle dos dados e rotas.

Com criação de rotas para manipulação das requisições e respostas das páginas de modo dinâmico.

Uso de models e partials para melhoria da eficiencia do código.

(Tudo utilizando Javascript)


Front:

Utilizado Html5 + css3 + Bootstrap 4 para criação da page index e perguntas.

As outras pages foram inseridas de modo dinâmico através das rotas feito pelo backend em nodejs e express.

Utilização de formulário para serem utilizadas como parâmetro das pages dinâmicas.



Database:

Utilizado Mysql + workbench para visualização dos dados em sql.

Tabelas - colunas - Linhas.

Utilizado Sequelize para uso dos models (Sequelize permite inserir, alterar, deletar ou atualizar dados em sql com o uso da linguagem Javascript).
